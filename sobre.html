Sobre o Game Hub

🎮 O que é o Game Hub?

O Game Hub é uma plataforma gratuita de jogos online, criada para reunir jogos classicos e divertidos em um unico lugar. Todos os jogos rodam diretamente no navegador, sem necessidade de download ou instalacao. Basta acessar, escolher um jogo e se divertir!

👨‍💻 Quem criou?

O Game Hub foi desenvolvido por Igor Sales Teixeira, com o objetivo de praticar desenvolvimento web e criar algo divertido e acessivel para todos.

🛠️ Tecnologias utilizadas

O projeto foi construido usando tecnologias web modernas:

  • HTML5
  • CSS3
  • JavaScript
