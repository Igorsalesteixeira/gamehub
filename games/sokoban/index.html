Sokoban

Empurre as caixas ate os alvos!

Como jogar Sokoban Online

Sokoban é o clássico puzzle japonês de empurrar caixas. Mova o personagem pelas setas ou botões, empurrando as caixas até os alvos marcados no chão. A dificuldade está no planejamento — você só pode empurrar, nunca puxar. Use o botão Desfazer se errar. Jogue sokoban online grátis.

Regras

  • Use as teclas de seta ou os botões na tela para mover o personagem.
  • O personagem empurra caixas ao andar na direção delas — certifique-se que há espaço atrás.
  • Coloque todas as caixas sobre os alvos (marcados com X ou cor diferente) para completar o nível.

Dicas

  • Nunca empurre uma caixa para um canto sem alvo — ela ficará presa para sempre.
  • Use o botão Desfazer livremente e planeje cada movimento com antecedência.
