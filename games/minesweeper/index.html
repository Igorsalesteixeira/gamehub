Como jogar Campo Minado Online

Campo Minado (Minesweeper) é o clássico jogo de lógica onde você deve revelar todas as células seguras do tabuleiro sem clicar em nenhuma mina. Os números indicam quantas minas estão nas células adjacentes. Escolha entre fácil, médio e difícil — jogue campo minado online grátis.

Regras

  • Clique em uma célula para revelá-la; se for mina, você perde.
  • Os números mostram quantas minas existem nas 8 células ao redor.
  • Clique com o botão direito (ou use o botão Bandeira) para marcar uma mina suspeita.

Dicas

  • O primeiro clique nunca é mina — clique no centro para abrir mais área.
  • Se o número de bandeiras ao redor de uma célula numerada coincidir com o número, as demais células vizinhas são seguras.
