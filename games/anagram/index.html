Como jogar Anagrama Online

Anagrama é um jogo de palavras onde você recebe letras embaralhadas e deve descobrir a palavra original. Clique nas letras na ordem correta para formá-la antes do tempo acabar. Com categorias variadas e 10 rodadas por partida, é o jogo ideal para treinar vocabulário. Jogue anagrama online grátis.

Regras

  • Clique nas letras embaralhadas na ordem correta para formar a palavra original.
  • Use a categoria exibida como dica temática para descobrir a palavra.
  • Três dicas disponíveis por partida revelam uma letra na posição correta cada.

Dicas

  • Identifique prefixos e sufixos comuns (-ção, -dade, -mente) entre as letras disponíveis.
  • Use o botão Embaralhar para ver as letras em uma nova ordem — às vezes ajuda a visualizar a palavra.
