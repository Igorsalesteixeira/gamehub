Como jogar Pac-Man Online

Pac-Man é um dos jogos mais icônicos de todos os tempos. Navegue pelo labirinto comendo todos os pontinhos enquanto foge dos 4 fantasmas coloridos. Coma as frutas grandes (power pellets) para ficar temporariamente poderoso e devorar os fantasmas assustados. Jogue pac-man online grátis sem download.

Regras

  • Use as setas, WASD ou os botões na tela para mover o Pac-Man pelo labirinto.
  • Coma todos os pontinhos para completar o nível; pontinhos grandes dão poderes especiais.
  • Encostar em um fantasma (sem poder especial ativo) custa uma vida.

Dicas

  • Aprenda os padrões de movimento dos fantasmas — cada um tem comportamento diferente.
  • Use os corredores laterais para escapar rapidamente de situações difíceis.
